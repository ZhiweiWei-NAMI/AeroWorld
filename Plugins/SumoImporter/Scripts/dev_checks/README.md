# Dev Checks

This folder is for temporary or analyst-facing inspection scripts.

Examples:

- inspect one UAV across sampled ticks
- inspect capture sidecars for framing or altitude issues
- inspect facility placement, scale, and visibility against the active capture view
- verify render-ready rosters, truth frames, and dynamic labels preserve semantic fields
- compare generated truth against runtime sidecars

If a script becomes part of the normal workflow, move it up into `Plugins/SumoImporter/Scripts/` or into `donghu_core/`.
